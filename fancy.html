Section 1 — Preparation

The Nintendo Wii isn't a hard unit to modify, and other guides are usually straight-forward about this topic. This will also walk you through other software that may be of interest to any user with a modified Nintendo Wii, so you can enjoy the full benefit of this process with enhancements to the console's functionality.

If for some reason you forgot your parental control code, there is a website which allows you to remove this code rather easily. If this happens to be the case, follow these steps:

  1. Make sure the time and date on the Wii accurately reflects your console's time, relative to its timezone,
  2. Have the console know you forgot / don't know the console's parental control code.
  3. Visit this website to input the confirmation code you would otherwise give to Nintendo.
  4. Use the unlock code that website gave you; this should help to unlock the Wii, should you not know its parental control code.
After following these steps, you are free to do whatever you please with the console, which in the case of this guide is to install third-party applications.

Section 3 — Homebrew initialization

Option 1 — Letterbomb (Offline method)
This is only for if your Wii cannot purpose the Internet channel, or is unable to connect to the Internet for some other reason.

For this method to work, you will need to obtain a special archive made specifically for your system. All you need is as follows:

  • An SD card 32GB or less
  • An active Internet connection with another device

In order to begin with letterbomb, you need to find your console's Machine Access Code (MAC). To find your MAC, navigate to the following sections in order:

  1. Wii Settings
  2. Internet (on page 2)
  3. Console Information

From there, you will see a series of six hexadecimal numbers: This is the number you need to input at https:/.please.hackmii.com. At that website, do the following:

  1. Specify your console's region
  2. Input your console's MAC address
  3. Verify you are human, then click either of the submission buttons.
  4. Download, then extract the specially-crafted archive to the root of your Wii's SD card.
  5. Eject the SD card, then insert into the Wii.

After the above, it is simply a matter of visiting the message board and if applicable, looking in yesterday's messages for a red envelope with a bomb inside. This will execute savezelda, which will forward you to the Homebrew Channel Installer.

Option 2 — FlashHax (Online method)
For this to work, installation of the Internet Channel is required; if not, get it from the Wii Shop Channel.
If the console crashes at payload acquisition, then power off, confirm your connection to the outside world is good and try again.

Unlike the offline method, this is way easier:

  1. Open the Internet Channel
  2. Choose to input an address (the button labelled "WWW")
  3. Go to flashhax.com
  4. Select region of console from this webpage
  5. Favourite the page your selection lead you to.
  6. From favourites, select the page you just bookmarked.

When the payload finishes downloading, savezelda should execute and forward you to the Homebrew Channel Installer. Installation of the Homebrew Channel, then exiting this software will forward you to the Homebrew Channel where you can perform instructions shown in the next section.

Section 4 — System configuration
If nothing else, install BootMii. It is a rather important piece of kit.

Required — Homebrew Channel

This is pretty straight-forward: You choose to install the Homebrew Channel after the summary and proceed with installation. There are no special requirements beyond what you needed to do for accessing the Homebrew Channel installer for installing this utility, which will be required later for additional steps.

Recommended — BootMii
While technically not a requirement, multiple software reference it so excluding this will most likely lead to malfunction or crashing of the system should an option to access this tool is available and chosen by accident without the software for it to reference.

The Homebrew Channel Installer does more than just install the Homebrew Channel: it also bundles with it a powerful system memory management suite known as BootMii and provide multiple options for installing it, depending on what hardware is inside the Wii. Depending on the console's Boot2 hardware, how BootMii can be installed will be indicated with text and colour. Colours as follows:

  • Green: BootMii installable to Boot2
  • Pink: BootMii can only be executed from SD card
  • Red: Uninstallable (Always the case with Wii U / vWii.)

If you can, install BootMii in Boot2. If not, then proceed with IOS installation. This means the SD card is required for BootMii to function. There will be additional instructions on configuring how it boots shown later.

If this software is excluded, Homebrew Channel will not list it. This software only exists when presence of IOS 254 is detected.

Recommended — IOS236 Installer
Older versions of this software would lock you out of your Wii for a time if you told the software your intent was piracy when prompted.
If no other IOS works for manipulating system memory in any way, this one will in most circumstances.

This software installs a patched version of IOS 36 as a separate entity; IOS 236. In some cases, this may be the only IOS you can use to write new data onto the Wii via Multi-Mod Manager, which is why this is included.

When prompted, you will be given a choice to install additional components for piracy. This is a trick question, and due to Dr. Clipper's previous actions with older versions of this installer I had to include the most recent version, as all telling that version of the software you're a pirate does is undo everything it did, forcing you to run the installer once again.

Recommended — Priiloader
Without this software, should the system menu become inaccessible for any reason you will be soft bricked. There's a way out of a soft brick, but it is beyond your means without additional (and rather bespoke) hardware to do so, requiring assistance of whatever community members possess these tools if you are unable to acquire them yourself.

Priiloader is an awesome utility which once installed, can boot before the system menu by holding down Reset while powering on from either idle or standby. While Priiloader's main purpose is to act as another layer of protection against soft bricks by providing a means into BootMii and The Homebrew Channel, you can also perform the following;

  • Load applications before anything else, aside from BootMii
  • Install software to later specify for automatic boot
  • Use various hacks for the console's functionality (via hacks_hash.ini)
  • Password-protect its settings (defined with password.txt in apps\priiloader)

All of these functions and more will be purposed later on, for the time being this covers installation; just run the installer and follow the prompts displayed. That's it.

Making channels use RC24

This is the most involved of the processes. As follows:

  1. Access the Wii System Menu
  2. Open the Wii configuration menu, lower left
  3. Open Wii settings
  4. Go to page 2, then select Internet
  5. Select Connection Settings, then pick your active Internet connection
  6. Choose to change its settings
  7. Scroll through available options until option Auto-Obtain DNS
  8. Set primary DNS to the following: 164.132.044.106
  9. Set secondary DNS to any of the following:
    • Your ISP's DNS address, per router, modem or endpoint settings (probably slowest)
    • Google's DNS: 8.8.8.8 or 8.8.4.4
    • IBM's DNS (Quad9): 9.9.9.9 or 149.112.112.112
    • Cloudflare's DNS: 1.1.1.1 or 1.0.0.1

Nintendont
This is part of wiikit_ugly.zip because it can encourage cheating, and is only really useful with a utility which facilitate USB loading. Get it from section 2.

Nintendont is a USB loader for Gamecube. It can even permit the use of the Gamecube system menu, should one wish and allow for cheating, if enabled in Nintendont's settings. Along with just the program itself, the copy of Nintendont provided also brings with it a variety of themes for menus and when loading titles, so after configuring these settings the software will look more-or-less how you wish.

You may use Nintendont in conjunction with Dios Mios, a replacement MIOS for Gamecube if you wish however unlike Dios Mios, Nintendont won't break compatibility with Gamecube media should you choose to use Gamecube discs.

WiiFlow

WiiFlow is a USB loading utility with a coverflow-style menu. With an appearance inspired by Apple's iTunes which had CoverFlow and a variety of themes available via the Internet, WiiFlow can be modified to become a very personalized and elegant way of displaying your digital library. On top of being a USB loader for Wii and Gamecube titles (with the included build being specially optimized for use with Nintendont), it can act as a replacement or supplement for the following:

  • The Homebrew Channel
  • Wii System Menu
  • Nintendont / Dios Mios
  • Various emulators via plugins
  • WiiRD (cheats database)
  • CleanRip (for copying games)

It is a very useful utility, as seen from its capabilities to supplement or displace tools which you may otherwise need to obtain for copying and managing media. The only thing it cannot do is install WADs, but this is what HacksDen's Multi-Mod Manager is for. Using the items from section 5 as an example, you can install just about any WAD representative of WiiWare titles for loading those from system memory as well.

Section 7 — Wii software for PC

Wii Backup Manager
Due to what this software is capable of, it is contained in wiikit_ugly.zip. Get it from section 2.
This software does not work for Gamecube ISOs. There is another utility to shrink Gamecube games shown in the next option.

This software is used to manage Nintendo Wii titles. While not entirely necessary to enjoy a Nintendo Wii title loaded from USB, it provides several key benefits:

  • Easier management of installed titles
  • Handling of media on two separate devices (or locations)
  • Conversion of titles into several formats

The last of the listed highlights is important to understand, as obtaining Wii titles from the Internet presently isn't as easy as Wii U and 3DS titles, which is subject to change in the near future. As we aren't there now, a suite of tools are required to use content from the Internet via USB media or SD card.

The WBFS format should be of interest for most people who are archiving a lot of titles as it features the main benefits of compact ISO (CIOS) while allowing the titles to be usable in all USB loaders. However, some software for PC requires the title to be of ISO format before changes can be made, which would decompress the image and return it to original file size.

From observing features presented in this software, it appears like there is a possibility of using original media to make backups on PC, however not all optical devices are compatible which if this is the case for your PC, tools like CleanRip (not covered) or WiiFlow (from wiikit_ugly.zip) should be used on the Wii instead.
Always keep the original media you are modifying until confirming changes made do not render modified media unplayable. There is a distinct possibility some titles will not function at all and for the sake of saving a few megabytes for larger titles, the effort may not be worth it.

WIimmfi patch tools
This is part of wiikit_ugly.zip because the act of patching using these tools can be considered as piracy even if you own the original without the effort of using a USB loader or dedicated software for creating an archive of the intended target. Even with the original copy, modifying games with these tools can still be considered violation of domestic and international copyright laws.
This software only works on ISO files, and most may work even if they were scrubbed. Use Wii Backup Manager to convert WBFS to ISO.

Being the most popular of the two WFC alternatives (the other, not covered being AltWFC), Wiimmfi (pronounced whim-fi) is very easy to implement into any WFC-compatible game. Below is a summary of the directories provided:

  • ACCF: Patcher designed specifically for Animal Crossing: Let's Go to the City / City Folk.
  • MKWii: Patcher designed specifically for Mario Kart Wii.
  • Generic: Patcher for all other Wii WFC games, such as Super Smash Bros. Brawl or Bomberman Blast.
  • WiiWare: patcher for all WFC WiiWare titles, such as Uno or Tetris Party.

The patchers provided can be used as follows:

  1. Place compatible images inside of their respective directories.
    • If there are system-specific directories, place images in there, assume further instructions are followed there.
  2. Run the patcher batch or shell file, respective for your operating system
    • If Windows SmartScreen warns about the file being potentially malicious, it isn't; Click on More Info and sselect Run anyway
    • If on an open-source system or Mac, you may need to execute `chmod a+x` on the file before executing the shell script in terminal.
  3. Collect the modified ISO image at the directory the script generates

Once modified, you should be able to launch your game and use the WFC functionality it originally had, thus retaining the original value of the titles affected. For a full list of compatible titles before you begin modding, check out this compatibility list.

While there are a lot of "Compatible" titles, few ever garnered massive communities. Online experience may be worse for lesser-known titles, requiring a significant interest by the Wiimmfi community before fast matchmaking is a possibility.
